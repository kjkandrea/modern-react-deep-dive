# 렌더링은 어떻게 일어나는가?

렌더링 : 모든 컴포넌트들이 현재 자신들이 가지고 있는 props 와 state 의 값을 기반으로 어떻게 UI를 구성하고 어떤 DOM 결과를 제공할 것인지 계산하는 일련의 가정

* 최초 렌더링 : 처음 페이지에 진입하면 발생
* 리렌더링 : 
  * `setSate` 가 실행되는 경우
  * 클래스형 컴포넌트의 `forceUpdate` 가 실행되는 경우
  * `useReducer` 의 `dispatch` 가 실행되는 경우
  * `key props` 가 변경되는 경우
  * `props` 가 변경되는 경우
  *  부모 컴포넌트가 렌더링 될 경우
